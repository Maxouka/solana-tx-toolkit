What's in the toolkit?

Priority Fee Optimizer

Dynamic fee estimation with percentile-based recommendations (p25/p50/p75/p90). Queries getRecentPrioritizationFees and computes optimal fees for any network condition.

Rust TypeScript
Jito Bundle Builder

MEV-aware transaction submission with bundle construction, tip optimization, retry logic, and exponential backoff. Supports up to 5 atomic transactions per bundle.

Rust Jito SDK
Jupiter Route Optimizer

Integrated Jupiter V6 aggregation with slippage protection, multi-hop routing, and route comparison. Best execution across all Solana DEXs.

TypeScript Jupiter V6
Transaction Monitor

Real-time transaction tracking with WebSocket subscriptions and polling fallback. Event-driven architecture with status progression callbacks.

TypeScript WebSocket
